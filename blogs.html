What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, allowing developers to manipulate the content, structure, and style of HTML or XML documents through scripting languages like JavaScript.

How do you select an element from the DOM?

Select an Element: You can select an element from the DOM using methods like document.getElementById() to find an element by its ID, document.querySelector() for a CSS selector, or document.getElementsByClassName() to select elements by their class name.

What is event delegation in the context of the DOM, and why is it useful?

Event Delegation: Event delegation is a technique where a single event listener is added to a parent element to manage events for multiple child elements. This approach improves performance and simplifies code, especially when handling events for dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM?

Manipulate Attributes and Styles: You can manipulate an element's attributes using setAttribute() to add or modify an attribute, and removeAttribute() to remove it. For styles, use element.style.property to dynamically change CSS properties, such as element.style.color = 'red' to change text color.
